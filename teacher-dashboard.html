Create logins for your students

Choose how many you need — usernames and passwords are generated for you. You can add more any time.

How the custom start works
  1. Leave it blank for random, easy-to-read usernames (e.g. otter482) — the default.
  2. Or type a class code, e.g. Y10K.2026 — letters, numbers, dots, hyphens and underscores only.
  3. Your students are then numbered from it: S1Y10K.2026, S2Y10K.2026, S3Y10K.2026 …
  4. Generating more later with the same code carries the numbering on (it won't repeat S1, S2…) — safe to reuse across sessions or even other classes.

Every student in this batch will share this password until they change it themselves — students can set their own from Manage Account → Password once they've signed in.
