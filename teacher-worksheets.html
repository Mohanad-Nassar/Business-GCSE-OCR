How it works: ① pick questions from any topics (same picker as Tasks) → ② choose layout, word-bank and SEN options → ③ preview, then Print / Save as PDF (or download an editable Word copy). Nothing is assigned to students — this only makes paper. The answer key always starts on a fresh final page, so print “all pages except the last” for the class set. Questions stay in sync with the topic pages automatically because they come from the same live question bank as tasks.

Worksheets aren't tied to a class — pick which subject's question bank to build this one from. Switching subject clears any questions already chosen below.

2Worksheet options

✏️ Fill-in-the-blanks & word banks

Blanks questions are grouped into rounds, each with its own word bank, so one giant bank never overwhelms students. Fewer questions per round + fewer extra words = easier (good for SEN); one big round + many extra words = harder.
